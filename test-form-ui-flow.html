Test Form Creation Flow

Steps:

  1. Open browser console (F12)
  2. Go to Create New Form
  3. Enter form name: "Test KH-G1-Level1"
  4. Click "Apply Template" button
  5. Select "KH-G1-Level1" template
  6. Check all activities you want to include
  7. Click "Apply Template" in dialog
  8. Check console for logs about sections and fields
  9. Click "Save" button
  10. Check console for saving logs

What to look for in console:

    - "Applying template with sections: ..."
    - "Total fields: ..."
    - "Saving form: ..."
    - "Form sections count: ..."

Expected vs Actual:

Expected: Multiple sections with multiple fields each

Actual: Form saves with only one field
